# Struuuuudel music

A collection of songs, experiments, and sketches created while learning Strudel.

## About

This repository documents my journey learning Strudel, a live coding environment for creating music in the browser. It includes recreations of existing songs, original ideas, and small experiments as I explore different techniques and patterns.


## Goals

- Learn the fundamentals of Strudel
- Recreate songs to understand different techniques
- Experiment with rhythms, melodies, and effects
- Document interesting patterns and discoveries

## Running

Open the files in the Strudel editor and evaluate them to play the patterns.
